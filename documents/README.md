# Documents

Placez ici les documents téléchargeables pour le club :

## Fichiers requis

### 1. fiche-inscription.pdf
Fiche d'inscription au club de badminton de Sochaux.

**Contenu suggéré :**
- Nom, prénom
- Date de naissance
- Adresse
- Email et téléphone
- Certificat médical (copie à joindre)
- Signature et date

### 2. decharge-mineur.pdf
Autorisation parentale pour les mineurs non accompagnés.

**Contenu suggéré :**
- Identité du mineur
- Identité des parents/tuteurs légaux
- Autorisation de pratique en autonomie
- Autorisation de transport
- Numéro d'urgence
- Signature des parents et date

## Note

Les liens de téléchargement sont configurés dans `index.html` (section #documents).

Si vous n'avez pas encore ces documents, vous pouvez :
1. Les créer avec un traitement de texte (Word, LibreOffice)
2. Les exporter en PDF
3. Les placer dans ce dossier avec les noms exacts ci-dessus
